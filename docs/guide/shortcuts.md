# Keyboard Shortcuts

`cliedit` supports a wide range of keyboard shortcuts for efficient editing. Many follow standard conventions found in modern editors.

## General

| Key | Action |
| --- | --- |
| `Ctrl+S` | **Save** and Exit |
| `Ctrl+Q` | **Quit** without Saving (Prompts if changes exist) |

## Navigation

| Key | Action |
| --- | --- |
| `Arrow Keys` | Move cursor one character/line |
| `Alt+Left` / `Alt+Right` | Jump by **Word** |
| `Home` / `End` | Jump to Start / End of Line |
| `Page Up` / `Page Down` | Scroll Up / Down by one screen |
| `Ctrl+L` | **Go to Line** (Enter number) |
| `Ctrl+M` | **Match Bracket** (Jump to matching `()`, `[]`, `{}`) |

## Editing

| Key | Action |
| --- | --- |
| `Type` | Insert character |
| `Backspace` | Delete character before cursor |
| `Delete` | Delete character after cursor |
| `Enter` | Insert new line |
| `Tab` | Insert soft tab (spaces) or **Indent Selection** |
| `Shift+Tab` | **Outdent Selection** |
| `Ctrl+K` | **Cut Line** (Delete current line) |
| `Ctrl+U` | **Paste Line** (Insert previously cut line) |
| `Ctrl+D` | **Duplicate** current line or selection |
| `Alt+Up` / `Alt+Down` | **Move Line(s)** Up / Down |

## Selection

| Key | Action |
| --- | --- |
| `Ctrl+Arrow Up/Down` | Select text by line (vertically) |
| `Ctrl+Arrow Left/Right` | Select text by character (horizontally) |

## Clipboard

| Key | Action |
| --- | --- |
| `Ctrl+C` | **Copy** Selection (or All if nothing selected) |
| `Ctrl+X` | **Cut** Selection |
| `Ctrl+V` | **Paste** Selection |

## Search & Replace

| Key | Action |
| --- | --- |
| `Ctrl+W` | **Find** (Enter query) |
| `Ctrl+R` | **Replace** (Enter query, then replacement) |
| `Ctrl+G` | **Find Next** occurrence |

In Search/Replace mode:
- `Enter`: Confirm / Next Step
- `Esc` / `Ctrl+C`: Cancel

## History

| Key | Action |
| --- | --- |
| `Ctrl+Z` | **Undo** |
| `Ctrl+Y` | **Redo** |
